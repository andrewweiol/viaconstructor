dxfpreview

DXF-Previewer and to image cobverter for thumbnail generation

Usage:

$ dxfpreview -h
usage: __main__.py [-h] [-x WIDTH] [-y HEIGHT] [-o OUTPUT] filename

positional arguments:
  filename              dxf file

optional arguments:
  -h, --help            show this help message and exit
  -x WIDTH, --width WIDTH
                        screen width
  -y HEIGHT, --height HEIGHT
                        screen height
  -o OUTPUT, --output OUTPUT
                        save to image

Examples:

display

$ dxfpreview tests/data/simple.dxf

convert to image

$ dxfpreview tests/data/simple.ngc -x 320 -y 240 -o simple.jpg
